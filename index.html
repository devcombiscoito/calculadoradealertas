Somador de Alerta

Bem-vindo ao Somador Interativo!

Esta é uma ferramenta simples e elegante desenvolvida em HTML, CSS e JavaScript. Nosso objetivo é demonstrar a funcionalidade básica de interação com o usuário através de alertas (alert()) e entradas de dados (prompt()).

Como funciona?

Ao recarregar a página, um script JavaScript é executado imediatamente. Ele solicitará que você insira dois números. Após a inserção, um alerta exibirá a soma exata dos valores fornecidos.
